# MiprimerBaseDatosJava
mi primer base de datos realizada el 22 de mayo de 2026
este proyecto funciona con el mysql-connector-j-9.6.0.jar y xampp
Necesita una base de datos llamada alumnos, con una tabla llamada talumnos con los siguientes indicaciones:
+----------+--------------+------+-----+---------+----------------+
| Field    | Type         | Null | Key | Default | Extra          |
+----------+--------------+------+-----+---------+----------------+
| clave    | mediumint(9) | NO   | PRI | NULL    | auto_increment |
| nombre   | varchar(30)  | YES  |     | NULL    |                |
| edad     | int(11)      | YES  |     | NULL    |                |
| estatura | float        | YES  |     | NULL    |                |
+----------+--------------+------+-----+---------+----------------+
